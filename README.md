Лабораторная работа.
Калькулятор поддерживает 4 операнда, приоритеты операций и разные виды округления.

## Требования
Для запуска исходного кода требуется:
- Node.js (версия 18 или выше)
- Angular CLI (`npm install -g @angular/cli`)

## Инструкция по установке и запуску (Source Code)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git
Перейдите в папку проекта:
code
Bash
cd big-number-calculator
Установите зависимости (обязательно, так как node_modules не хранятся в git):
code
Bash
npm install
Запустите локальный сервер разработки:
code
Bash
ng serve
Откройте в браузере: http://localhost:4200
